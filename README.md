# Health-risk-predictor2
Ai based biomedical health risk detector app
## 🎯 Aim

To develop a biomedical web application that analyzes basic human vital parameters and predicts the health risk level of a patient using a rule-based approach.
## ⚙️ Procedure

1. Collect basic patient data such as:
   - Age
   - Heart Rate
   - Body Temperature
   - Oxygen Saturation (SpO2)

2. Input the data into the web application interface.

3. The system processes the data using predefined medical thresholds.

4. The algorithm evaluates the condition based on:
   - SpO2 levels (respiratory health)
   - Heart rate (cardiac condition)
   - Body temperature (infection/fever)

5. The system classifies the patient into:
   - Normal
   - At Risk
   - Critical

6. The result is displayed instantly to the user.
